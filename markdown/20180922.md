+ How to test Python project with specific version in Mac OS X?

<!-- PELICAN_END_SUMMARY -->

Travis-ci
===

Test with Ubuntu and Mac OS X platform.

<pre class="brush: yaml">
# .travis.yml

language: python

matrix:
  include:
    
    - os: linux
      sudo: required
      python: 3.6
      dist: xenial
    
    - os: linux
      sudo: required
      python: 3.7
      dist: xenial
    
    - os: osx
      osx_image: xcode10
      language: generic
      env: PYTHON=36
</pre>

Travis-ci are not provide Python language in "osx" platform, so we choose "generic".
